Install Recipe Grabber

A one-click bookmark that imports the recipe you're reading.

Drag this button onto your bookmarks bar:

Grab Recipe

Then visit any recipe page and click it.

No bookmarks bar showing?

  1. Chrome / Edge: Ctrl+Shift+B (or ⌘+Shift+B on Mac) to toggle it.
  2. Firefox: right-click the toolbar → Bookmarks Toolbar → Always Show.
  3. Safari (Mac): View menu → Show Favorites Bar.

Can't drag? Paste it as a bookmark URL instead.

Click to copy.

  1. Right-click your bookmarks bar → Add page… (or use Ctrl+D, then choose to edit).
  2. Name it Grab Recipe, paste the copied text as the URL, save.

Tap to copy the bookmarklet URL to your clipboard.

  1. Tap the orange button above to copy.
  2. Bookmark this page: tap Share ⬆️ → Add Bookmark → Save.
  3. Open Bookmarks (the open-book icon) → Edit.
  4. Tap the bookmark you just made. Change the name to Grab Recipe and replace the URL by pasting (tap-and-hold the URL field → Paste).
  5. Tap Done. You're installed.

Catch: If pasting silently strips javascript: off the front, position the cursor at the start of the URL and type those 11 characters in manually.

Using it

  1. Open any recipe page in Safari.
  2. Tap the Grab Recipe bookmark.
  3. A new tab opens with the editable recipe form. The original page stays on screen until the import finishes.

First time only: if nothing happens, go to Settings → Safari → Block Pop-ups and turn it OFF (or allow pop-ups for recipes.tbotb.com).
